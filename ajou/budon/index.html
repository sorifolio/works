AIBiS.on:
AI Biosignal Solution

생체신호(심전도, 뇌파 등)를 수집할 수 있는 플랫폼

  • 중환자실, 수술실, 응급실 등에서 수집되는 생체신호(심전도, 뇌파 등)를 수집할 수 있는 플랫폼입니다.
  • GE, Philips, Nihon Koden 사의 환자 모니터 기기와 연동 가능합니다. (타기종 모니터도 추가 개발 가능)
  • 고밀도의 대용량 데이터 수집 인프라로, 생체신호 기반 인공지능 연구를 지원합니다.
  • 아주대 생체신호 데이터 수집 현황: http://52.78.14.28:3000/
  • 생체신호 데이터 뷰어: http://52.78.14.28:8080/
  • (인공지능 기반 SaMD*가 탑재된 버전은 현재 개발 중입니다)
    *SaMD: Software as a Medical Device

PtoD.on:
Pixel Image to Data Solotion

이미지 형태의 자료에서 유용한 데이터를 추출해주는 솔루션

  • 이미지 형태의 자료에서 유용한 데이터를 추출해주는 솔루션입니다.
  • 병원 전자의무기록 내에는 이차 활용이 불가능한 스캔된 보고서 형태의 자료들이 많이 존재합니다. (청력검사, NST 검사, 폐기능 검사 등)
  • 인공지능 기술을 이용하여 이미지 형태의 자료에서 유용한 정보를 인식하고 이를 데이터로 추출하게 됩니다.
  • 본 솔루션을 통해 보유 중인 데이터의 가치를 업그레이드 시킬 수 있습니다.
  • 현재 개발 완료된 적용 사례
    • 청력검사 보고서 내 그래프의 수치화
    • NST 검사 결과지의 파형 정보 추출 및 전산화
